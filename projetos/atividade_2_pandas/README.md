# Atividade 2: Limpeza, Transformação e Agrupamento de Dados

Este diretório contém a resolução da segunda atividade prática de **Ciência de Dados**, focada na manipulação avançada de dados reais utilizando a biblioteca **Pandas**.

## Sobre o Projeto
O objetivo desta atividade é realizar uma análise exploratória e tratamento de dados sobre o dataset de **Emendas Parlamentares**. Foram aplicadas técnicas para responder a perguntas de negócio através da manipulação de DataFrames, garantindo a integridade e a clareza das informações extraídas.

## O Dataset
* **Ficheiro de Dados:** `emendas_transformadas.json` (ou `.parquet`)
* **Contexto:** Dados detalhados sobre a aplicação de recursos públicos via emendas, incluindo ano, tipo de emenda, autor, localidade e valores (empenhado, liquidado, pago).

## Estrutura de Ficheiros
* **`pandas_intro_ex2.ipynb`**: Notebook contendo o código Python com as soluções passo a passo.
* **`emendas_transformadas.json`**: Base de dados utilizada na análise.
* **`README.md`**: Documentação detalhada da atividade.

## Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Biblioteca Principal:** Pandas
* **Ambiente:** VS code / Google Colab / Git Bash
