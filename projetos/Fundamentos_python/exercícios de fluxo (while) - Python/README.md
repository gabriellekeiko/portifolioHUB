# Estruturas de Repetição (Loop While)

Este diretório contém os exercícios práticos desenvolvidos no **Spyder**, focados na implementação de laços de repetição controlados por condições lógicas em Python.

## Sobre os Exercícios
O loop `while` é fundamental para criar programas que precisam repetir um bloco de código enquanto uma condição específica for verdadeira. Nesta seção, foram explorados:
* **Repetição com Condição:** Execução de blocos baseada em estados lógicos dinâmicos.
* **Contadores e Acumuladores:** Técnicas para contagem de iterações e soma de valores durante o ciclo.
* **Validação de Dados:** Uso do loop para garantir que o utilizador insira informações corretas antes de prosseguir.
* **Critérios de Parada:** Implementação de sentinelas e interrupções controladas para evitar loops infinitos.

## Estrutura de Ficheiros
* **`exercícios_de_fluxo_(while)_-_Python.py`**: Script contendo a aplicação prática de loops para resolução de problemas iterativos.
* **`README.md`**: Documentação detalhada da seção.

## Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **IDE:** Spyder (Scientific Python Development Environment)
* **Conceitos:** Iteração Condicional e Controle de Fluxo.

## Como Executar
1. Abra o ficheiro no **Spyder**.
2. Certifique-se de que o **IPython Console** está ativo.
3. Pressione **F5** para iniciar a execução.
4. Interaja com o programa através do console (os loops `while` geralmente dependem de inputs para serem finalizados ou para mudar de estado).
