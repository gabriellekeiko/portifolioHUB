Este diretório centraliza os exercícios práticos desenvolvidos durante o **Bootcamp de Ciência de Dados (CEUB)** utilizando a IDE **Spyder**. O objetivo desta seção é consolidar o domínio da sintaxe Python e a lógica de controle de fluxo.

## Objetivos de Aprendizagem
* Compreensão de tipos de dados e variáveis.
* Implementação de estruturas de decisão e repetição.
* Manipulação de listas e sequências.
* Desenvolvimento de algoritmos para resolução de problemas matemáticos e lógicos.

# 🔁 Estruturas de Iteração (Loop For)

Este diretório contém os exercícios práticos desenvolvidos no **Spyder**, focados na implementação de laços de repetição definidos sobre sequências em Python.

## 📌 Sobre os Exercícios
O loop `for` é utilizado quando conhecemos previamente o número de iterações ou quando desejamos percorrer os itens de uma coleção (como listas ou strings). Nesta seção, foram explorados:
* **Iteração sobre Sequências:** Percurso de listas, tuplas e strings de forma eficiente.
* **Função range():** Geração de intervalos numéricos para controlo de ciclos.
* **Acumuladores e Filtros:** Processamento de dados dentro do loop para gerar somatórios ou filtragens específicas.
* **Aninhamento de Loops:** Uso de estruturas `for` dentro de outras para resolução de problemas matriciais ou tabuadas.

## 📂 Estrutura de Ficheiros
* **`exercícios_de_fluxo_(for)_-_Python.py`**: Script contendo a aplicação prática de iterações para processamento de coleções e intervalos.
* **`README.md`**: Documentação detalhada da seção.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **IDE:** Spyder (Scientific Python Development Environment)
* **Conceitos:** Iteração Definida e Manipulação de Sequências.

## ⚙️ Como Executar
1. Abre o ficheiro no **Spyder**.
2. Garante que o **IPython Console** está ativo.
3. Pressiona **F5** para iniciar a execução.
4. Observa as saídas no console; este tipo de loop é ideal para automação de tarefas repetitivas com limites conhecidos.
