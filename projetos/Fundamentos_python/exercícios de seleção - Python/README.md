# Estruturas de Seleção (If, Elif, Else)

Este diretório contém os exercícios práticos desenvolvidos no **Spyder**, focados na implementação de estruturas condicionais em Python.

## Sobre os Exercícios
A lógica de seleção permite que o programa execute diferentes blocos de código com base em condições lógicas (Verdadeiro ou Falso). Nesta seção, foram explorados:
* **Condicionais Simples:** Uso do `if` para validações diretas.
* **Condicionais Compostas:** Uso do `else` para caminhos alternativos.
* **Condicionais Encadeadas:** Uso do `elif` para múltiplas verificações sucessivas.
* **Operadores Lógicos:** Aplicação de `and`, `or` e `not` para criar condições complexas.

## Estrutura de Ficheiros
* **`exercícios_de_seleção_-_Python.py`**: Script contendo a lógica de limpeza, tratamento de dados e filtragem utilizando estruturas de decisão.
* **`README.md`**: Documentação detalhada do projeto.

## Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **IDE:** Spyder (Ambiente de Desenvolvimento Científico)
* **Conceitos:** Lógica Booleana e Controle de Fluxo.

## Como Executar
1. Abra o arquivo no **Spyder**.
2. Certifique-se de que o **IPython Console** está visível.
3. Pressione **F5** para rodar.
4. Caso o programa solicite uma entrada (ex: "Digite sua idade"), responda diretamente no console à direita para ver a resposta do algoritmo.
