# Estruturas de Dados (Listas em Python)

Este diretório contém os exercícios práticos desenvolvidos no **Spyder**, focados na manipulação de coleções de dados utilizando **Listas** em Python.

## Sobre os Exercícios
As listas são estruturas de dados fundamentais em Python, permitindo armazenar, organizar e manipular sequências de elementos de forma dinâmica. Nesta seção, foram explorados:
* **Manipulação de Listas:** Criação, indexação e fatiamento (*slicing*) de elementos.
* **Métodos Nativos:** Utilização de funções essenciais como `.append()`, `.insert()`, `.remove()`, `.pop()` e `.sort()`.
* **Iteração com Listas:** Integração de listas com laços de repetição (`for` e `while`) para varredura e processamento de dados.
* **Operações de Agregação:** Uso de funções matemáticas integradas como `sum()`, `max()`, `min()` e `len()` sobre coleções numéricas.


## Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **IDE:** Spyder (Scientific Python Development Environment)
* **Conceitos:** Coleções Dinâmicas, Mutabilidade e Estruturas de Dados Sequenciais.

## Como Executar
1. Abra o ficheiro no **Spyder**.
2. Garanta que o **IPython Console** está ativo.
3. Pressione **F5** para iniciar a execução do script.
4. Acompanhe a manipulação dos dados diretamente no console ou utilize o **Variable Explorer** do Spyder para inspecionar os estados das listas em tempo real.
