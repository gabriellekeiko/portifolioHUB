# Exercícios de Lógica de Programação (Estruturas Condicionais)

Este diretório contém uma série de scripts desenvolvidos em **Python** utilizando a IDE **Spyder**. O foco destes exercícios é a aplicação de estruturas de decisão (`if`, `elif`, `else`) para a resolução de problemas lógicos.

## Sobre os Exercícios
Estes exercícios foram realizados como parte da minha formação acadêmica no **CEUB**. O objetivo é consolidar a base teórica de controle de fluxo, tratando diferentes condições e entradas de dados de forma dinâmica.

## Tecnologias e Ferramentas
* **Linguagem:** Python 
* **IDE:** Spyder (Ambiente de Desenvolvimento Científico)
* **Versionamento:** Git & GitHub
