# Exercícios de Estruturas de Repetição (Loop While)

Este diretório reúne scripts desenvolvidos em **Python** utilizando a IDE **Spyder**, focados na implementação de fluxos de repetição controlados por condições lógicas (`while`).

## Sobre os Exercícios
O objetivo desta seção é demonstrar o domínio sobre loops e iterações. Os exercícios abordam cenários onde a repetição de um bloco de código é necessária até que uma condição específica seja atingida, como validação de dados, contadores e acumuladores.

## Tecnologias e Ferramentas
* **Linguagem:** Python
* **IDE:** Spyder
* **Versionamento:** Git & GitHub
