# Atividade 2: Limpeza, Transformação e Agrupamento de Dados

Este diretório contém a resolução da segunda atividade prática de **Ciência de Dados**, focada em processos de **ETL (Extract, Transform, Load)** e manipulação avançada de grandes volumes de dados (Dataset de Emendas Parlamentares).

## Sobre o Projeto
Nesta etapa, o foco principal foi a preparação dos dados para análises complexas. Trabalhei com o dataset `emendas_tratamento.parquet`, aplicando técnicas de limpeza para tratar inconsistências e transformações para extrair métricas de valor, como agrupamentos por partido, autor e funções de governo.

## O Dataset
* **Fonte:** Dados de Emendas Parlamentares.
* **Formato Original:** Parquet (formato otimizado para Big Data).
* **Processamento:** O projeto culmina na exportação dos dados tratados para o formato **JSON**.

## Tecnologias e Ferramentas
* **Linguagem:** Python 3.11+
* **Biblioteca Principal:** Pandas
* **Formatos de Dados:** Parquet (leitura) e JSON (escrita/exportação)

## Competências Demonstradas
1. **Manipulação de Formatos Binários:** Leitura eficiente de ficheiros `.parquet`.
2. **Data Cleaning:** Identificação e tratamento de valores ausentes e inconsistentes.
3. **Engenharia de Atributos:** Criação e transformação de colunas para análise.
4. **Agrupamentos (Group By):** Realização de cálculos agregados (soma, média, contagem) por categorias específicas.
5. **Exportação de Dados:** Conversão do DataFrame final para ficheiros JSON para integração com outros sistemas.

## Instruções de Execução
1. Certifica-te de que o ficheiro `emendas_tratamento.parquet` está acessível ou no mesmo diretório.
2. Abre o ficheiro `pandas_intro_ex2.ipynb`.
3. Executa o passo a passo para visualizar o processo de transformação de dados até à célula final de download do JSON.
