# Atividade 1: Exploratory Data Analysis (EDA) com Pandas

Este diretório contém a resolução da primeira atividade prática de **Ciência de Dados**, focada no domínio dos comandos fundamentais da biblioteca **Pandas** para exploração inicial de dados.

## Sobre o Projeto
O objetivo desta atividade é realizar uma análise exploratória inicial (EDA) para entender a estrutura, o volume e a qualidade de um conjunto de dados. Foram aplicados comandos essenciais para extrair estatísticas descritivas e garantir que os dados estejam prontos para análises mais profundas.

## O Dataset
* **Ficheiro de Dados:** Utilizado para prática de comandos básicos (DataFrames).
* **Contexto:** Exploração de métricas, tipos de dados, contagem de valores únicos e identificação de registros duplicados.

## Tecnologias e Ferramentas
* **Linguagem:** Python 3.11+
* **Biblioteca Principal:** Pandas
* **Ambiente:** Git Bash / Google Colab / Vs Code

## Competências Demonstradas
1. **Carregamento de Dados:** Uso de `read_csv` ou para importar informações.
2. **Visualização Inicial:** Utilização de `head()` e `tail()` para inspeção de registros.
3. **Metadados:** Verificação de tipos de colunas e memória com `info()`.
4. **Qualidade de Dados:** Identificação de valores nulos e verificação de duplicatas com `duplicated().sum()`.
5. **Estatística Descritiva:** Geração de resumos numéricos com `describe()`.

## Instruções de Execução
1. Abra o ficheiro `pandas_intro_ex1.ipynb`.
2. Execute as células sequencialmente.
3. Leia as interpretações em Markdown abaixo de cada output para entender as conclusões extraídas dos dados.
