# Projetos de Análise de Dados com Pandas

Este diretório centraliza os meus projetos de manipulação, limpeza e análise exploratória de dados (EDA) utilizando a biblioteca **Pandas**. Aqui, aplico técnicas de Ciência de Dados para transformar dados brutos em insights estruturados e arquivos prontos para consumo.

---

## Objetivos do Módulo
O foco destas atividades é dominar o ciclo de vida dos dados, incluindo:
* **Ingestão:** Leitura de múltiplos formatos (CSV, JSON, Parquet).
* **Exploração (EDA):** Identificação de padrões, anomalias e estatísticas descritivas.
* **Limpeza (Data Cleaning):** Tratamento de valores nulos, duplicados e inconsistentes.
* **Transformação (ETL):** Agrupamentos, filtragens e criação de novas métricas de valor.

---

## Organização dos Projetos

Para facilitar a navegação, os projetos estão divididos por nível de complexidade:

### 1. Atividade 1: Fundamentos de EDA
Primeiro contato prático com manipulação de DataFrames.
* **Foco:** Comandos básicos (`info`, `describe`, `head`), verificação de tipos e identificação de duplicados.
* **Resultado:** Um diagnóstico completo da saúde inicial de um conjunto de dados.

### 2. Atividade 2: ETL e Tratamento de Emendas
Projeto avançado focado no dataset de Emendas Parlamentares.
* **Foco:** Manipulação de ficheiros Parquet, tratamento de dados ausentes e exportação final para JSON.
* **Destaque:** Uso de `groupby` para extrair somas e médias por categorias políticas e governamentais.

---

## Stack Técnica
* **Linguagem:** Python 3.x
* **Bibliotecas:** Pandas, NumPy.
* **Ambientes:** Vs Code, Git Bash e Google Colab.
* **Formatos:** Manipulação de Big Data com `.parquet` e integração com `.json`.

---

## Metodologia Aplicada
Em todos os notebooks deste diretório, sigo o padrão de **Código + Interpretação**. Para cada operação técnica realizada, existe uma célula de Markdown explicando o impacto dessa mudança nos dados e o que os resultados significam para o negócio ou pesquisa.
