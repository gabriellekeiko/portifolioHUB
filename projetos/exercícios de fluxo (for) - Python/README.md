# Exercícios de Estruturas de Iteração (Loop For)

Este diretório contém scripts desenvolvidos em **Python** utilizando a IDE **Spyder**, focados na iteração sobre sequências (listas, intervalos e strings) utilizando a estrutura `for`.

## Sobre os Exercícios
Diferente do loop `while`, estes exercícios focam na iteração definida. O objetivo é demonstrar o uso de contadores, percurso de coleções de dados e a função `range()`, competências essenciais para a automação de tarefas e processamento de listas.

## Tecnologias e Ferramentas
* **Linguagem:** Python 
* **IDE:** Spyder
* **Versionamento:** Git & GitHub

## Estrutura de Ficheiros
* **`exercicios de fluxo (for) - Python`**: Script contendo a lógica de iteração, percurso de listas e uso de acumuladores com a estrutura for.
* **`README.md`**: Documentação detalhada da seção.

## Como Executar os Exercícios no Spyder
Para rodar os scripts desta pasta e validar a lógica de programação, siga os passos abaixo:
### 1. Pré-requisitos
É necessário ter o ambiente **Anaconda** instalado ou apenas a IDE **Spyder** isolada.
### 2. Execução Passo a Passo
1. **Abrir a IDE:** Inicie o Spyder através do Anaconda Navigator ou do menu iniciar.
2. **Carregar o Ficheiro:** Vá em `File > Open` e selecione o arquivo (ex: `exercicios_fluxo_for.py`).
3. **Configurar o Console:** Certifique-se de que o **IPython Console** está aberto (geralmente no canto inferior direito).
4. **Rodar o Código:** - Pressione **F5** ou clique no ícone de "Play" (seta verde) na barra de ferramentas superior.
   - Caso o exercício solicite dados (como `input()`), digite os valores diretamente no console e pressione **Enter**.
### 3. Dica de Visualização
Para acompanhar as variáveis criadas durante a execução (como contadores de loops ou acumuladores), utilize a aba **"Variable Explorer"** no painel superior direito do Spyder. Isso facilita a compreensão do fluxo de dados em tempo real.
